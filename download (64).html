      Java - Java Deployment and Best Practices

       At the end of this module, you should be able to:
      • convert our JAVA Spring applications from using H2 to using PostgreSQL
      • deploy to a cloud service a full RESTful Web Services application with a JAR using PostgreSQL, Maven, Spring including Spring Boot, and Tomcat
      • connect to a JavaScript Frontend to authenticate to a Java Spring Application including determining authenticated users information
      • use CURL to perform CRUD operations using a Spring Application
      • understand what else a Java programmer should know
      • locally deploy a full RESTful Web Services application with a JAR using PostgreSQL, Maven, Spring including Spring Boot, and Tomcat (*** OPTIONAL ***)

      Overview

      The Java Unit you are completing

      • gives you a solid foundation in Java and the Spring Framework including a good foundation upon which to build future knowledge.
      • prepares you for being a backend developer for Lambda School’s Build Projects and Labs.
      • covers just over half of the topics that interest our hiring partners.

      To be a successful Java Backend developer, you will need to continue your Java Spring Framework studies. Below are topics gathered from several hiring partners, topics that we ran out of time to cover in the Java Course. The topics are roughly in order of importance, that is how often employers said they wished for someone who knew the topics. Lambda School is committed to providing a high-quality thorough education enabling you to obtain the best possible job! We are working on creating our own materials to cover these topics. Until they are available I have provided some external resources that you can use as a good starting point for studying the given topic. As always, do feel free to reach out to your instructions and fellow students for help!

      Note that not all topics are the same. Learning about immutability and final might take an hour or so. Learning AWS could easily be a full Lambda School unit. Good Luck and keep learning!!!
